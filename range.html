Understanding the Range

The **range** of a data set is the difference between the highest and lowest values. It is a measure of how spread out the values in a data set are.

Steps to Find the Range

To find the range, follow these steps:

Step 1: Identify the highest and lowest values in the data set.
Step 2: Subtract the lowest value from the highest value to find the range.

Example: Finding the Range

Example 1: Find the range of the numbers 4, 8, 3, 12, 7, 10.

Step 1: The highest value is 12 and the lowest value is 3.
Step 2: Subtract 3 from 12: 12 - 3 = 9.
Range = 9

Example with Different Set:

Example 2: Find the range of the numbers 15, 22, 30, 10, 18.

Step 1: The highest value is 30 and the lowest value is 10.
Step 2: Subtract 10 from 30: 30 - 10 = 20.
Range = 20
